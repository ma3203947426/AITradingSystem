Core Agent Flow

Market Data Fetch A-share quotes, indices and K-line history through akshare.
Indicators Calculate MA, EMA, MACD, RSI, Bollinger, ADX, ATR and volume ratios.
Voting Combine trend, MA crossover, RSI and Bollinger signals with confidence weights.
Risk Control Check drawdown, loss streaks and position size before an action is allowed.
Execution Run paper trades, persist journal records and trigger strategy evolution.

Built Components

  • Data feed module for A-share market data.
  • Strategy engine with backtesting and grid search optimization.
  • Enhanced decision engine with multi-strategy weighted voting.
  • Paper trader for local portfolio, PnL and journal management.
  • Dashboard generator for HTML visualization.

Why Agent-driven

The system turns scattered manual trading tasks into a repeatable loop. Each cycle observes market state, reasons over strategy signals, checks portfolio risk, records evidence, and uses historical outcomes to decide whether the strategy version should evolve.

This project is for technical research and paper-trading demonstration only. It does not provide investment advice.
